# Commands

You can click on the `PilotCodi` status bar item to open the [PilotCodi Command Palette](command:pilotcodi.commandPalette.trigger) and select the desired command.

![PilotCodi Command Palette](./commandPalette.png)

You can also search for `PilotCodi` in the [command palette](command:workbench.action.quickOpen?%5B%22%3EPilotCodi%22%5D) to see a list of all available commands provided by the PilotCodi extension.

**Note**: Commands related to chat features are only visible when your server supports them.
