# Connect to Server

To connect to the PilotCodi server, follow these steps:

1. Click the PilotCodi status bar item to open the commands panel and select [Connect to Server...](command:pilotcodi.connectToServer).

2. Follow the instructions to input the PilotCodi server's endpoint URL:
   ![connectToServer_url](./connectToServer_url.png)

3. Enter the token:
   ![connectToServer_token](./connectToServer_token.png)

Once the connection is established, the PilotCodi status bar item will show a checkmark:

![statusbarReady](./statusbarReady.png)

## Cross-IDE Settings

If you are using multiple IDEs, you can use the [PilotCodi Agent Config File](command:pilotcodi.openPilotCodiAgentSettings) to share the settings across IDEs. Refer to the [Online Documentation](https://pilotcodi.pilotcodiml.com/docs/extensions/configurations/) for more details.
