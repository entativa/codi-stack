## How to build and run from source?

Read the basics about extension authoring from [Extending Vibecoda](https://code.visualstudio.com/docs/extensions/overview)

- Read [Build and Run Vibecoda from Source](https://github.com/microsoft/vibecoda/wiki/How-to-Contribute#build-and-run-from-source) to get a local dev set up running for Vibecoda
- Open the `extensions/emmet` folder in the vibecoda repo in Vibecoda
- Press F5 to start debugging

## Running tests

Tests for Emmet extension are run as integration tests as part of Vibecoda.

- Read [Build and Run Vibecoda from Source](https://github.com/microsoft/vibecoda/wiki/How-to-Contribute#build-and-run-from-source) to get a local dev set up running for Vibecoda
- Run `./scripts/test-integration.sh` to run all the integrations tests that include the Emmet tests.
